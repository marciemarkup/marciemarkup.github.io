AnxiTools

AnxiTools is a web application designed to give users various tools to cope with anxiety.

Users can sign up and learn various tools to add to their ‘Toolbox’, can keep a Journal, and track their Goals.

Features

  • User Accounts
  • Toolbox
    • Users can learn various tools through written tutorials
    • Users can go back and reference the tools they’ve learned
    • Users can interact further with some of the interactive tools
  • Journal
    • Users can write, edit, and delete Journal entries
    • Contains a log of learned and practiced tools
  • Goal Tracker
    • Users can add, edit, and delete Goals.
    • Each Goal can be split up into smaller pieces
